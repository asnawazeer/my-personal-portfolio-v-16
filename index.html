<!DOCTYPE>

Defines the document type and version of HTML.

<!DOCTYPE>

<html>

The root element of an HTML page.

<html lang="en">...</html>

<head>

Contains metadata, title, and linked resources.

<head>... </head>

<title>

Sets the page title (seen on browser tab)

<title>My Page <title>

<body>

Contains all visible page content.

<body>...</body>

<h1> to <h6>

Heading tags.<h1> is the largest, <h6> is the smallest.

<h1>Main Heading<h1>

<p>

Paragraph tag used to write text.

<p>This is a paragraph.<p>

<a>

Anchor tag used for links.

<a href="https://example.com"><Visit/a>

<img>

Used to display images.

<img src="image.jpg" alt="My Image">

<br>

Line break tag. No closing tag.

This is line 1<br>This is line 2

<hr>

Horizontal line. Used to separate content.

<hr>

<ul,> <ol,> <li>

Used for unordered and ordered lists.

<ul>  
    <li> Item 1 </li>
    <li> Item 2 </li>
</ul>

<ol>
    <li> First </li> 
    <li> Second </li>
</ol>

<table>

Used to display tabular data.

<table>
    <tr><th>Name</th><th>Age</th></tr>      
    <tr><td>Ali</td><td>20</td></tr>
</table>

<form>

Used to collect user input.

<form action="#" method="post">
  ...
</form>

<input>

Used for various input types like text, radio, checkbox.

<input type="text" name="name">

<label>

Used to label form inputs.

<label for="email">Email:</label>

<textarea>

Multi-line input for text.

<textarea rows=4 cols=50></textarea>

<select name="course">
    <option>HTML</option>
    <option>CSS</option>
</select>
